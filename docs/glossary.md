# อภิธานศัพท์ (Glossary)

| คำ | ความหมาย |
|---|---|
| **ใบเสร็จบริจาค** (donation) | เงินบริจาค 1 ก้อนที่มีใบเสร็จรับเงิน 1 ใบ — หน่วยหลักของระบบ แต่ละใบทำหน้าที่เป็น "ซองเงิน" ที่ตัดรายจ่ายออกได้จนหมด |
| **เลขที่ใบเสร็จ** (receipt_no) | เลขใบเสร็จของมหาวิทยาลัย เช่น `RC1055845` — ห้ามซ้ำทั้งระบบ |
| **วันที่ในใบเสร็จ** (receipt_date) | วันที่ออกใบเสร็จ — **ใช้เป็นเกณฑ์แยกปีของทุกรายงาน** (ปี พ.ศ. ม.ค.–ธ.ค.) |
| **วันที่บริจาค** (donated_date) | วันที่เงินเข้าบัญชีจริง อาจคนละปีกับวันที่ในใบเสร็จ — เก็บไว้อ้างอิง ไม่ใช้แยกปี |
| **วัตถุประสงค์ในการบริจาค** (purpose) | ข้อกำหนดการใช้เงินตามที่ผู้บริจาคระบุ: ข้อ 1, ข้อ 2, ข้อ 3, ข้อ พรก., ข้อไม่ระบุวัตถุประสงค์, ข้อ อื่นๆ (เพิ่มได้ในหน้าตั้งค่า) |
| **รหัส FD13** | รหัสกองทุนของมหาวิทยาลัย เช่น `N0045`, `N0142` — เป็นรายการอ้างอิงแยกจากวัตถุประสงค์ **ไม่ผูกกัน** |
| **หมวดหมู่** (category) | หน่วยงาน/โครงการผู้รับเงิน เช่น ภาควิชา, E-LU Lab — เพิ่มใหม่ได้บ่อยจากหน้าตั้งค่า |
| **รายจ่าย** (expense) | การใช้เงิน 1 ครั้ง เช่น "ทุนการศึกษา เดือนมิถุนายน" มียอดจ่ายรวม 1 ยอด |
| **เลขที่ส่งออก** (doc_no) | เลขหนังสือส่งออกของรายจ่าย เช่น `ท.13/69` |
| **การตัดเงิน** (allocation) | การระบุว่ารายจ่ายก้อนหนึ่งใช้เงินจากใบเสร็จใบไหน เท่าไหร่ — รายจ่าย 1 รายการตัดจากหลายใบเสร็จได้ ผลรวมต้องเท่ายอดจ่าย |
| **ยอดคงเหลือ** (balance) | ยอดใบเสร็จ − ผลรวมการตัดเงินของใบนั้น — คำนวณสดเสมอ ไม่มีการเก็บค่า (จาก view `donation_balances` หรือ `donations_list_view` แล้วแต่หน้า) |
| **ปี** ในรายงาน | ปี พ.ศ. ของวันที่ในใบเสร็จ (รายรับ) / วันที่จ่ายเงิน (รายจ่าย) |
| **"ทั้งหมด"** ในตัวกรองปี | ตัวเลือกพิเศษในดรอปดาวน์เลือกปี (ค่า default) — แสดงข้อมูลรวมทุกปีที่มีในระบบ แทนที่จะกรองเฉพาะปีเดียว ใช้ได้ทั้งแดชบอร์ด, รายงาน, รายการบริจาค, รายจ่าย |
| **ช่องทางบริจาค** (channel) | วิธีรับเงิน เช่น โอนเงิน, เงินสด |
| **ข้อมูลบัญชี** (account) | บัญชี/ระบบที่เงินเข้า เช่น EDONATION, กสิกรไทย |
| **การกรองตั้งแต่ keystroke** | กติกา input ตัวเลขของระบบ: ตัวอักษร/เครื่องหมาย/หลักเกินพิมพ์ไม่ติดตั้งแต่แรก (ไม่ใช้ `type="number"` — ดู ADR-006) ช่วง พ.ศ. ที่ยอมรับ 2500-2600, จำนวนเงิน ≤ 9 หลัก ทศนิยม 2 ตำแหน่ง |
