# ADR-003: ไม่มีทะเบียนผู้บริจาคแยก — เก็บชื่อบนใบเสร็จโดยตรง

สถานะ: ยอมรับแล้ว (12 ก.ค. 2569)

## บริบท

ระบบ DDEE เดิมบังคับให้ "เพิ่มรายนามผู้บริจาคใหม่" ก่อน แล้วจึงเพิ่มยอดบริจาคได้ —
เป็นหนึ่งในสาเหตุที่ผู้ใช้บอกว่า "ใช้งานยาก ขั้นตอนเยอะ"
ข้อมูลจริงใน Excel ก็เก็บแค่ชื่อ-นามสกุลบนแถวใบเสร็จ ไม่มีการใช้ที่อยู่/เลขบัตร ปชช.
ในรายงานใด ๆ

## การตัดสินใจ

- ไม่มีตาราง `donors` — เก็บ `donor_name` เป็นข้อความบนตาราง `donations` ตรง ๆ
- ฟอร์มมี autocomplete จากชื่อที่เคยกรอก (ค้นจากใบเสร็จเดิม) เพื่อให้สะกดตรงกัน

## เหตุผล

- เพิ่มใบเสร็จจบในฟอร์มเดียว — ตัดขั้นตอนที่เป็นปัญหาของระบบเดิม
- รายงาน "รายนามผู้บริจาค" ยังทำได้ด้วย GROUP BY donor_name

## ผลที่ตามมา / ข้อแลกเปลี่ยน

- ชื่อสะกดต่างกันจะนับเป็นคนละคน — บรรเทาด้วย autocomplete
- ถ้าอนาคตต้องออกหนังสือขอบคุณ/ใบลดหย่อนภาษีที่ต้องใช้ที่อยู่
  ค่อยเพิ่มตาราง donors แล้ว backfill จากชื่อ (ทำได้ไม่ยาก)
